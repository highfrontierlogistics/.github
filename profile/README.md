# High Frontier Logistics 🚀

**The orbital tender .blend is public.**

[Download it from github.com/wescorp/orbital-tender](https://github.com/wescorp/orbital-tender) — the full geometry, rigging, and animation pipeline. Open it, fork it, build on it.

---

A vessel that accepts, stores, services, and redeploys Starship-class payloads. Less "spaceship," more "container port that happens to be in orbit." One person designed it end to end. The source files are public now.

**The drop:** geometry, rigging, animation pipeline, and the thesis behind it. Not renders — the actual source files. Everything you'd need to pick up where I left off, adapt the design, or build your own argument on top of it.
